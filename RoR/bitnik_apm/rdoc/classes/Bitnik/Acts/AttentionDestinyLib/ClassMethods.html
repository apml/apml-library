acts_as_attention_destiny(options = {:key_mapping => nil, :from_value => 'default'})

Defines this model to be an attention destiny. An attention destiny is equivalent to a <Concept> in the APML specification. This method will associate an attention_destiny method that gives access to the AttentionDestiny object associated to the model. Params:

key_mapping
a method that will be called to generate the ‘key’ attribute of the APML concept element. By default, the plugin will look for a key attribute in the model.
from_value
the value for the from attribute of the APML from concept. If nothing is specified, the value will be set to ‘default’.
